## <a name="whats-translated"></a>翻譯內容

應用程式使用您的語言偏好顯示如慣用語言表達的功能表、標籤文字和系統訊息元素。

因為大部分的功能都是根據您的資料而定，所以部分項目仍然是以原始語言提供。 例如：

- 您的資料，包括實體和區段
- 您的環境名稱

## <a name="choose-your-language-settings"></a>選擇您的語言設定
